Modal Light Box

To view a specific window by highlighting it over and above other contents.
Auto Hide Sticky Note

This is an example for Basic Modal Box you just need to insert "basic-modal" Class to use this Modal Box and you can easily edit its contents.

$('.basic-modal').click(function (e) {
$('#basic-modal-content').modal();
return false;
});

Basic Modal Dialog


For this demo, SimpleModal is using this "hidden" data for its content. You can also populate the modal dialog with an AJAX response, standard HTML or DOM element(s).


Examples:


$('#basicModalContent').modal(); // jQuery object - this demo

$.modal(document.getElementById('basicModalContent')); // DOM

$.modal('<p><b>HTML</b> elements</p>'); // HTML

$('<div></div>').load('page.html').modal(); // AJAX

Fixed Sticky Note

This is an example for Scrolling Modal Box you just need to insert "scrolling-moda" Class to use this Modal Box and you can easily edit its contents.

$('.scrolling-moda').click(function (e) {
$('#basic-modal-content2').modal();
return false;
});
