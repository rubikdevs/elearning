Responcive Popup Form Alert

This is an example for "Simple Alert" you just need to insert "simple-alert" ID to use this alert.

$.msgbox("In order to process your request you must provide the following:", {
    type    : "prompt",
    inputs  : [
      {type: "text",     label: "Insert your Name:", value: "George", required: true},
      {type: "password", label: "Insert your Password:", required: true}
    ],
    buttons : [
      {type: "submit", value: "OK"},
      {type: "cancel", value: "Exit"}
    ]
  }, function(name, password) {
    if (name) {
      $.msgbox("Hello "+name+", your password is "+password+".", {type: "info"});
    } else {
      $.msgbox("Bye!", {type: "info"});
    }
  });
